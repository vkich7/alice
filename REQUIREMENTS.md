# ✅ Вимоги до HTML-сторінки

## 1. Структура HTML-документа

Файл `index.html` повинен містити:

```html
<!DOCTYPE html>
<html lang="uk">
<head>
    ...
</head>
<body>
    ...
</body>
</html>
```

| Елемент | Вимога |
|---------|--------|
| `<!DOCTYPE html>` | Обов'язково першим рядком |
| `<html lang="uk">` | Атрибут мови — `uk` |
| `<head>` | Присутній |
| `<body>` | Присутній |

---

## 2. Мета-теги у `<head>`

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Назва розділу — Аліса в Задзеркаллі</title>
<link rel="stylesheet" href="style.css">
```

| Елемент | Вимога |
|---------|--------|
| `<meta charset="UTF-8">` | Обов'язково |
| `<title>` | Непорожній, містить назву розділу |
| `<link rel="stylesheet" href="style.css">` | Посилання на зовнішній CSS |

---

## 3. Обов'язкові елементи у `<body>`

### `<h1>` — Заголовок сторінки
- Рівно **один** тег `<h1>` на сторінці
- Містить назву розділу (наприклад, "Задзеркалля" або "Живі квіти та комахи")

### `<p>` — Параграфи тексту
- **Щонайменше 3** теги `<p>`
- Кожен абзац тексту — окремий `<p>`
- Весь текст із `text/pages/page_XX.txt` має бути присутній

### Додатково (необов'язково, але заохочується)
- `<h2>` для підзаголовків
- `<blockquote>` для цитат
- `<em>` або `<strong>` для виділення тексту
- `<nav>` з посиланнями на сусідні сторінки

---

## 4. Файл `style.css`

Файл повинен:
- Існувати в папці поряд з `index.html`
- Бути непорожнім
- Містити **щонайменше 5 CSS-правил**

### Обов'язкові властивості

| Властивість | Де застосовувати |
|-------------|-----------------|
| `font-family` | Будь-де (body, p, h1...) |
| `color` | Будь-де |
| `background-color` або `background` | body або будь-який блок |
| `margin` або `padding` | Будь-де |
| `font-size` або `line-height` | Будь-де |

### Приклад мінімального CSS

```css
body {
    font-family: Georgia, serif;
    background-color: #fdf6e3;
    color: #333;
    margin: 0;
    padding: 20px;
}

h1 {
    font-size: 2em;
    color: #8b0000;
}

p {
    line-height: 1.8;
    margin-bottom: 1em;
}
```

---

## 5. Відповідність тексту

Текст у `<p>` тегах повинен відповідати вмісту файлу `text/pages/page_XX.txt`:
- Текст не має бути скорочений більш ніж на 20%
- Перші та останні речення тексту мають бути присутні

---

## ❌ Типові помилки

- Забули `<!DOCTYPE html>`
- `charset` написали неправильно: `utf-8` замість `UTF-8` (обидва варіанти прийнятні, але саме `charset` обов'язковий)
- CSS-файл підключений, але порожній
- Текст вставлений без тегів `<p>`, просто в `<body>`
- Кілька `<h1>` на одній сторінці
